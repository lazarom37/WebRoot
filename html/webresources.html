  1. w3schools.com
    A large wikia-like resource for all things related to Web Development imagineable
    A free educational site made for learning coding online. Offers extensive tutorials, templates, and test environments to help anyone trying to learn any aspect of Web Development
    Rating: 5/5

  2. Codeacademy
    Interactive resource for learning code for several languages
    An online service that offers free and completely interactive classes for coding in several languages such as HTML, CSS, Java, JavaScript, C#, C++, SQL, React and more. Offers a paid Pro experience for personalized projects, quizzes and learning plans.
    Rating: 5/5

  3. git - the simple guide
    Quick resource for learning Git/Github
    A small webpage providing short and sweet guidance on how to use Git. Provided with large and easy to read text and buttons, highlighted commands that are important, and diagrams explaining processes such as workflows and branching. Very helpful all-in-one page for when you need to remember specific important commands and what they do
    Rating: 3/5

  4. Unsplash
    Free resource for stock images.
    A site that provides free and new stock images every day. Because of how important photos are in web design, especially during planning when there are no images to work with just yet, Unsplash provides a convenient and elegant resource for web developers in need of visually-pleasing placeholders
    Rating: 4/5

  5. w3 HTML Validator
    Site that checks the syntax of your HTML code
    HTML generally doesn't mind if you make egregious errors on your own, but when other browsers and attached script files are thrown into the mix it can create some "invisible" errors that your IDE wouldn't be able to find. w3's HTML Validator is a certified and reputable service that helps find unaccounted errors organized line-by-line on suggestions for a fix.
    Rating: 4/5
